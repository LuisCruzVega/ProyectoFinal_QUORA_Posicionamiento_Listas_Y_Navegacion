¿Cómo vinculan los desarrolladores de front-end su trabajo al back-end?

Por medio de una API (application program interface) existen de varios tipos, API SOAP y API REST.

Las APIs son interfaces en un servidor web que esperan por peticiones que se les envíe desde el frontend. Está interfaz posee endpoints que son algo como http://tusitio.com/users (obtener todos los usuarios registrados). Cada endpoint posee un verbo (GET, POST…)

Una vez recibas la respuesta del back, puedes renderizar esa respuesta en el front.

Es una explicación muy superficial pero te doy una idea para que profundices

¿Cuál es la diferencia entre desarrollo/tecnologías front-end y back-end?

Front-end es diseño gráfico, es el diseño de la experiencia del usuario. Back-End es pura lógica y tratamiento de datos. Front-End se enfoca en el usuario final, Back-end se enfoca en aplicar algoritmos en operaciones CRUD, mientras que el Front-end trata de que la complejidad en las operaciones quede siempre en el lado del Back-end. Mientras mas complejo y automatizado el Back-end, mas simple se vuelve el Front-end.

Ahora si te refieres a los lenguajes como tal, piensa en Excel vs PowerPoint o en hacer un biscocho vs decorar la torta con crema.
